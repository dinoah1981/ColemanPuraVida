# 🌴 Costa Rica 2026 — Coleman / Noah Family Trip

Interactive trip planner for our family vacation: **Feb 18–25, 2026**

## Setup

```bash
npm install
npm run dev
```

## Deploy to Netlify

1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → "Add new site" → "Import an existing project"
3. Connect your GitHub repo
4. Build settings (auto-detected):
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
5. Deploy!

### Add password protection

In Netlify dashboard → Site settings → Access & security → Visitor access → Set password.
